## Why

현재 '놀이' 섹션에는 매칭 게임과 분류 놀이만 있어, 아이들이 자음과 모음을 직접 조합하여 한글 글자를 만들어보는 체험이 없다. '글자 만들기' 놀이를 추가하여 한글의 조합 원리(초성+중성, 초성+중성+종성)를 자연스럽게 익힐 수 있게 한다.

## What Changes

- '놀이' 메뉴에 '글자 만들기' 유형 추가 (PlayTypes에 새 항목)
- **방식 1 - 기본 글자 만들기**: 왼쪽에 자음 카드, 오른쪽에 모음 카드를 배치하고, 각각 하나씩 선택하면 가운데에 합쳐진 글자(초성+중성)가 표시되며 TTS로 읽어줌
- **방식 2 - 받침 글자 만들기**: 방식 1과 동일한 레이아웃에 받침(종성) 선택 영역이 추가되어 초성+중성+종성 조합 가능, 완성 시 TTS로 읽어줌
- 두 방식 모두 완성된 글자 목록을 화면 하단에 최대 5개까지 표시하고, 탭하면 다시 소리를 재생
- 글자 만들기는 카테고리 선택 없이 바로 방식 선택 화면으로 이동

## Capabilities

### New Capabilities
- `letter-builder`: 자음+모음(+받침) 조합으로 한글 글자를 만들고 소리를 들을 수 있는 놀이 기능. 방식 1(초성+중성)과 방식 2(초성+중성+종성) 두 가지 모드 포함.

### Modified Capabilities
_(없음 - 기존 스펙 변경 없음)_

## Impact

- **라우팅**: 글자 만들기 방식 선택 및 각 방식별 새 라우트 추가 (`/play/letter-builder/modes`, `/play/letter-builder/basic`, `/play/letter-builder/advanced`)
- **컴포넌트**: `src/components/play/` 에 새 컴포넌트 3개 추가 (모드 선택, 기본 모드, 고급 모드)
- **데이터**: 기존 `jamo.json`의 자음/모음 데이터 활용, 받침용 자음 목록 추가 필요
- **오디오**: 조합된 글자의 TTS 재생 필요 - 브라우저 Web Speech API(`speechSynthesis`) 활용 (사전 녹음 파일 없이 동적 생성)
- **기존 코드**: `PlayTypes.tsx`에 새 놀이 유형 항목 추가, `routes.tsx`에 새 라우트 등록
